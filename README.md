# Get started with Azure Machine Learning Services
Welcome to this repository, this repository contains all the data, labs and 
slides for the workshop: Get started with azure machine learning services.

* **Goal for this workshop:** Get your first hands-on experience with Azure
  Machine Learning services.
* **Time needed:** 2 hours
* **Target audience:** Developers with a few years experience in C# or Java.

## Prerequisites
For this workshop you need to have an Azure Subscription. If you haven't got
access to a subscription you can [create a trial subscription](https://azure.microsoft.com/en-gb/free/).
